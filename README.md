# Deteksi Retak Beton Menggunakan CNN

Aplikasi ini dibuat menggunakan Streamlit dan TensorFlow untuk melakukan klasifikasi gambar beton retak dan tidak retak.

## Fitur

* Upload model CNN (.h5)
* Upload gambar beton
* Prediksi otomatis
* Menampilkan hasil klasifikasi dan tingkat keyakinan

## Cara Menggunakan

1. Upload file model `.h5`
2. Upload gambar beton
3. Sistem akan menampilkan hasil prediksi

## Teknologi

* Streamlit
* TensorFlow
* NumPy
* Pillow

## Deploy

Aplikasi dapat dijalankan menggunakan Streamlit Cloud atau secara lokal dengan:

streamlit run app.py
